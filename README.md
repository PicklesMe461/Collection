## LED Pins
8 Leds
GP2 to GP9

## Button Pins
GP14 Blue Button (Left for motors) PULL DOWN
GP22 White Button (Right for motors) PULL UP

## Pot Pin
GP26 Analog Input

## Servo Pin
GP0 Servo PWM Out

## DC Motor Pins
GP18 and GP17 PWM Out

## Step Motor Pins
GP10 GP11 GP12 GP13 PWM Out

## QRD1114 Sensor Pins
GP27 and GP28 Analog Input
